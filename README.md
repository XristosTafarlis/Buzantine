# The Buzantine Quest

Παιχνίδι πτυχιακής με θέμα την βυζαντινή ιστορία.

Game based on Byzantine History for my thesis.

## **Keybinds**
- Movement 
  > Walk : W A S D
  >
  > Run : Left Shift
- Attack
  > Left click
- Join Battle
  > Space Bar


- ***Cheats***
  > Plus (+) : Adds XP
  >
  > Backspace : Instant win
